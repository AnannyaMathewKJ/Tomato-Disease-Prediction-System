# 🍅 Tomato Disease Prediction System

A Deep Learning-based web application that identifies diseases in tomato leaves using Computer Vision. 

## 🚀 Features
* **Instant Detection:** Upload an image of a tomato leaf to get an immediate diagnosis.
* **High Accuracy:** Trained using a Convolutional Neural Network (CNN) on the PlantVillage dataset.
* **User-Friendly UI:** Built with Django for a seamless web experience.

## 🛠️ Tech Stack
* **Backend:** Django (Python)
* **AI Model:** TensorFlow / Keras (.h5 model)
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite (Development)
